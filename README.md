# Render VLESS (WS + TLS) 部署模板

## 一键部署
[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/YOUR_GITHUB_USERNAME/render-vless)

## 客户端配置
- 协议：VLESS
- 地址：<你的 Render 分配域名>
- 端口：443
- UUID：你在环境变量 UUID 中的值
- 流控：无
- 加密：none
- 传输协议：ws
- 路径：`/ws`
- TLS：开启
